# HylianScan v0.8 - Release Summary

Date: 2026-06-10

## Summary

Version `v0.8` significantly expands HylianScan with improved TCP reconnaissance, protocol-aware service identification, JSON reporting, 
TLS intelligence, and multi-provider passive subdomain discovery.

## Key Changes

- Replaced the long passive Subfinder flag from `--subdomains` to `--subfinder`.
- Preserved `-s` as the short alias for Subfinder.
- Added `-a / --amass` for passive Amass discovery.
- Added combined passive discovery mode with `-s -a`.
- Added active provider display before passive enumeration starts.
- Added provider-aware passive JSON export with per-provider counts.
- Added final deduplicated passive result output.
- Added source attribution so JSON consumers can identify which provider reported each subdomain.
- Marked the project release version as `v0.8`.

## Provider Modes

```bash
python3 hylianscan.py example.com -s
python3 hylianscan.py example.com -a
python3 hylianscan.py example.com -s -a
```

## Output Behavior

- TXT output remains enabled for passive discovery.
- JSON output remains optional through `--json-output`.
- Provider result lists are normalized, deduplicated, and sorted.
- Final passive JSON results include both merged subdomains and a source map.
